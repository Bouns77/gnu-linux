#### Configuration de l'IP forwarding

-Pour activer IP forwarding edité le fichier :/etc/sysctl.conf et ajouter ou decomander les lignes suivantes :
    -IPV4 : net.ipv4.ip_forward = 1
    -IPV6 : net.ipv6.conf.all.forwarding=1
